## Setup

```sh
$ git clone git@github.com:azukiazusa1/sapper-blog-app.git
$ cd sapper-blog-app
$ npm i
$ npm run dev
```

## Build

```sh
$ npm run build
```

### lint & typecheck

```sh
$ npn run lint
$ npm run typecheck
```

### vrt

```sh
$ npm run test
```
